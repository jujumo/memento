== Symbols

[NOTE]
====
* in Windows, type kbd:[Alt+XXX] where `XXX` are the digits of ASCII code,
* in Ubuntu (and inkscape), go into text mode and type kbd:[Ctrl+Shift+U] and the 4-digits unicode.
====
