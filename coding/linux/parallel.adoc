== inputs

=== list of files
.>bash
[source,bash]
----
parallel echo ::: /*
----
Will echo  `/bin`, `/boot`, `/cdrom`, `/dev`, `/etc` ...

=== values
.>bash
[source,bash]
----
parallel echo ::: 1 2 3
----
Will echo `1`, `3`, `2`.

=== value combinations
.>bash
[source,bash]
----
parallel echo ::: 1 2 3 ::: a b
----
Will echo  `1 a`, `1 b`, `2 a`, `2 b`, `3 b`, `3 a`.

=== value pairs
.>bash
[source,bash]
----
parallel --link echo ::: 1 2 3 ::: a b
----
Will echo  `1 a`, `2 b`, `3 a`,
