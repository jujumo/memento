## paths

### Script current dir

[quote, stackoverflow]
____
Reliable way for a bash script to get the full path to itself.
____


```bash
pushd `dirname $0` > /dev/null
SCRIPTPATH=`pwd -P`
popd > /dev/null
```

### get basename

```bash
$ s=/the/path/foo.txt
$ echo ${s##*/}
foo.txt
$ s=${s##*/}
$ echo ${s%.txt}
foo
$ echo ${s%.*}
foo
```

## loop over files
```bash
for f in *.jpg
do
  convert ./"$f" ./"${f%.jpg}.pdf"
done
```
